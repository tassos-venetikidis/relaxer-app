# RELAXER APP

Relax by controlling your breathing. This simple app provides an ideal tempo to breathe in, hold, and breathe out that is conducive to the handling of stress. By leveraging CSS animations, it provides a graphical conductor to your breathing, and with the usage of vanilla JavaScript, lets you use it as long as you like!

Made with:

- Grassroots HTML
- Plain, old CSS
- Always dependable, vanilla JavaScript
